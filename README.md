## 機能
* 為替レートAPIを利用して為替レートを表示　　
* 注文する(POST): 情報入力後、「注文」ボタンをクリックすると注文リストに追加
* 注文履歴を見る（GET）: ページロード後、下段の注文リストが自動的に表示される
